CSS Psudo Element

CSS pseudo-element is used to style specified parts of an element.
For example, it can be used to:
1)Style the first letter, or line, of an element 2)Insert content before, or after, the content of an element


The following properties apply to the ::first-line pseudo-element:
. font properties
. color properties
. background properties
. word-spacing
. letter-spacing
. text-decoration
. vertical-align
. text-transform
. line-height
. clear
